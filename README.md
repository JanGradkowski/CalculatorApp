# CalculatorApp

A sleek, fully functional desktop calculator designed with a modern dark-mode interface inspired by the Apple ecosystem. Built entirely in Java, this project emphasizes precise arithmetic logic alongside a highly polished, custom-styled graphical user interface.

## 🚀 Features

* **Core Arithmetic Logic:** Handles standard mathematical operations with reliable, meticulously tested precision.
* **Apple-Inspired UI:** Features a meticulously crafted layout mirroring the clean aesthetics of macOS and iOS calculators.
* **Dark Mode Native:** Styled extensively with custom CSS to provide a comfortable, high-contrast dark theme.
* **Responsive Interactions:** Smooth button hover states and click events for a premium, native-feeling user experience.

## 🛠️ Technologies Used

* **Language:** Java
* **UI Framework:** JavaFX
* **Styling:** CSS

## ⚙️ Setup and Installation

### Prerequisites
* Java Development Kit (JDK) 11 or higher installed on your system.

### Running the Application

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/rpggamesstudio/CalculatorApp.git](https://github.com/rpggamesstudio/CalculatorApp.git)
   cd CalculatorApp
2. Compile and run:
   ./mvnw clean javafx:run
